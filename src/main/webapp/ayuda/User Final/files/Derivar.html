       Derivar

Esta opción es usada para enviar un documento a algún participante del proceso, sin embargo, también puede enviarse con copia a otros participantes. Para derivar un documento dar clic en el botón "Derivar" , se mostrará la siguiente pantalla:

Aquí se debe de seleccionar el usuario a quien se va a derivar el documento, se registra el rango de fecha límite para la atención del documento, se selecciona el usuario  a quien se va a copiar el documento (si es que lo hubiera), ingresamos el asunto del envío. En caso que se requiera que la persona a quien se le está enviando el documento se le asigne también como responsable de todo el expediente, entonces ponemos un check en la opción "Responsable del Expediente" .Cabe mencionar que el usuario responsable asignado debe ser un participante del proceso. Adicionalmente podemos añadir un motivo u observación y usando el botón Adjuntar  se puede añadir un archivo al documento.
