       Carpetas de Búsquedas

Esta opción es usada para almacenar las búsquedas, ordenándolas en carpetas. Para crear una nueva carpeta de búsqueda debe de desplegar la opción Carpetas de búsqueda del menú y seleccionar Nueva Carpeta, se le mostrará la siguiente pantalla:

Aquí se muestran los criterios de búsqueda tales como el número de expediente, el número de documento, el número de Mesa de Partes, el Concesionario, el nombre del cliente, el área Destino, el Proceso al que pertenece el documento, el Tipo de Documento, el Documento de Identidad del Correntista, el rango de fecha de creación del Documento, el rango de fecha de creación del Expediente y la dirección principal. Se le debe de asignar un nombre a la carpeta. Para guardar los cambios usamos el  botón "Crear" 


Para visualizar los documentos encontrados en la búsqueda seleccionamos la carpeta guardada del menú y se mostrará en la parte derecha el detalle de dicha búsqueda, tal como lo muestra la siguiente pantalla:
