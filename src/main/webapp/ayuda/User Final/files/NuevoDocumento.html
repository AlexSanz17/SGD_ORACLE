       Nuevo Documento

       Para crear un nuevo documento, hacemos clic en el botón "Nuevo Documento" y se nos mostrará la siguiente pantalla:   


Esta opción nos permite añadir un nuevo documento a un expediente existente (haciendo referencia a este) o puede crearlo a partir de un expediente nuevo.


Para hacer referencia a un expediente existente, se debe hacer clic en el boton de búsqueda de expedientes, el cual nos mostrara una ventana de búsqueda de expediente por diferentes criterios, caso contrario deberá ingresar el nombre del proceso al cual pertenece el documento a registrar, en este caso se muestra por defecto el área de destino y destinatario correspondiente al proceso seleccionado. En la sección "Documento" se registra el tipo de documento (que ya se encuentran definidos,  tales como Carta, Oficio, Memo, etc), asunto del expediente, fecha del documento y los datos pertenecientes a la plantilla del documento seleccionado. Para registrar el nombre del cliente, se debe seleccionar el tipo de documento (DNI, RUC u otro) e ingresar el número correspondiente, dependiendo del tipo de documento seleccionado se mostrará razón social o nombre del cliente, dirección y ubigeo. Finalmente se registran observaciones si las hubiera.  


Adicionalmente, se puede adjuntar archivos al documento haciendo clic en el  botón "Adjuntar".

Para enviar el documento pulsamos el botón "Registrar Documento"   y se enviará al destino ya definido, generándose el Nº de documento y/o expediente según el caso.
